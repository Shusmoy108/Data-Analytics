card.scala             --- The scala file that contains the card object defination and method. 
                           Also the file contains the main object which reads the input file and
                           checks the validity of the cards. The output will be shown in the console
                           as well as written in output.txt file.
number.txt             --- The input file that contains the card numbers. Must be in the same repository
                           where the card.scala located.
output.txt             --- Contains the card number and the validity status of the card.
program_screenshot.png --- Contains the screenshot of the running program and console with the given input.

Commands to Run the Program:

-----> scalac card.scala
-----> scala main
